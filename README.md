### What I Do

I design and deploy **end-to-end computer vision pipelines** for heavy industry — from camera selection and algorithm development to PLC integration and on-site commissioning. My systems handle real-time object detection, thermal monitoring, defect inspection, and safety automation in some of the harshest environments in manufacturing.

**Core expertise:** Object Detection (YOLO v5/v8) · Instance Segmentation (SOLOv2, SAM) · Thermal Imaging · OCR · Anomaly Detection · PLC/HMI Integration (Delta, Siemens) · Video Analytics
